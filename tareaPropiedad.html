Propiedades de CSS

La siguiente tabla muestra las propiedades de CSS utilizadas en el ejercicio "To-do app" :

Nombre de la propiedad Función Posibles valores
@importPermite importar hojas de estilo externas en la CSS.url('ruta'), none
font-familyDefine la fuente tipográfica que se utilizará en el texto.nombre de la fuente, genérico (sans-serif, serif, etc.)
backgroundEstablece el fondo de un elemento, que puede ser un color o una imagen.#F0EFE9, #50E3A4, #FFFFFF, url('imagen'), none
marginControla el espacio exterior alrededor de un elemento.0, longitud (px, %, em), auto
colorDefine el color del texto de un elemento.#FFF
displayEspecifica cómo se comporta un elemento en el flujo del documento, determinando si se muestra como bloque, en línea o si se oculta.block, inline, flex, none
justify-contentAlinea los elementos dentro de un contenedor flexbox a lo largo del eje principal.center, flex-start, flex-end, space-between, space-around
heightDefine la altura de un elemento en la página.175px, auto, longitud (px, %, em)
align-itemsAlinea los elementos en un contenedor flexbox a lo largo del eje transversal.center, flex-start, flex-end, stretch
max-widthLimita el ancho máximo que puede tener un elemento.768px, none, longitud (px, %, em)
widthEspecifica el ancho de un elemento.100%, longitud (px, %, em)
font-sizeControla el tamaño del texto dentro de un elemento.1.5em, longitud (px, %, em)
text-alignDetermina la alineación del texto dentro de un elemento.right, left, center, justify
font-weightDefine el grosor de la fuente utilizada en el texto.100, 300, 400, 500, 700, 900
paddingControla el espacio interno entre el contenido y el borde de un elemento.40px, 20px 0px, longitud (px, %, em)
border-radiusEspecifica el radio de las esquinas de un elemento.20px, longitud (px, %, em)
border-bottomDefine el borde inferior de un elemento, incluyendo grosor, estilo y color.1px solid #CECECE
border-topDefine el borde superior de un elemento con sus características.1px solid #CECECE
marginEspecifica el margen alrededor de un elemento para separarlo de otros.0 20px, longitud (px, %, em)
